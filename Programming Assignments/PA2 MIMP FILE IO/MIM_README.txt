Image: The class lets us create image objects and do mutator and access methods for it. The class also allows us to save and load images from and to files.

ImageFileFormatException: Is the exception that is sent whenever the image methods load and save fail.

MimpApp: Lets us open and close our mim files through a gui. It also lets us save and load our mim files. We can even save pngs as mims.
